# Roadmap

## Phase 1: Local picker

- `.grabRoot()`
- `.grab(...)`
- registry and snapshot JSON
- overlay rectangles
- tap-to-select
- copy ID/JSON/XCTest/source
- shake and Command-Shift-D toggles

## Phase 2: Better graph quality

- z-index/window-level support
- overlapping candidate stack
- lazy list behavior
- scroll-container context
- active route/screen context
- automatic UIKit/AppKit tree scrape fallback
- better accessibility trait mapping

## Phase 3: Remote workflow

- outbound WebSocket broker
- browser-based viewer
- session tokens
- live event stream
- screenshot stream
- remote flash/select/toggle commands

## Phase 4: Design-system integration

- Figma component key
- design token capture
- typography/color/spacing validation
- owner/team metadata
- localization key capture
- analytics event capture

## Phase 5: Automation and AI workflow

- generate XCTest selectors
- generate UI-test skeletons
- explain disabled/hidden UI from captured state
- compare current UI graph to expected design graph
- create bug reports with selected node + screenshot + app state
